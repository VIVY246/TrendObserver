# Trend Collector

## 目的

複数SNS・Steam等から流行把握に使えるデータを取得し、AIが分析しやすいJSONL/CSV形式で出力する。

## 現時点の方針

- 公開サービスではなく、自分用ツールとして開発する
- 無料で自作できる範囲を優先する
- ジャンルを選択できる
- 複数ジャンル・複数データソースに対応する
- AI分析しやすい正規化データを出力する
