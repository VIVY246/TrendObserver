# Roadmap

## Phase 1: 基盤

- 要件と出力仕様を固定する。
- `NormalizedTrendItem` を実装する。
- JSONL writer を実装する。
- CLI の基本形を作る。

## Phase 2: 初期データソース

- Steam adapter を追加する。
- YouTube adapter を追加する。
- Reddit adapter を追加する。
- サンプル fixture と unit test を追加する。

## Phase 3: 分析しやすさの改善

- CSV writer を追加する。
- ジャンル別キーワードを拡充する。
- AI分析用のサンプルプロンプトを整備する。

## 後回し

- Web UI
- スケジューラー
- DB保存
- ダッシュボード
