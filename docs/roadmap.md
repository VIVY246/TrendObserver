# Roadmap

## Phase 1: TikTok CSV MVP

- Python CLI の土台作成
- TikTok CSV 取り込み
- 共通フォーマット正規化
- SQLite 保存
- AI 分析用 JSONL 出力
- AI 分析結果 JSONL 取り込み
- Markdown 週次レポート生成

## Phase 2: 入力と分析補助の拡張

- CSV 列名の揺れへの対応
- ジャンル YAML の拡充
- AI 分析用プロンプト例の整備
- レポートテンプレートの改善
- エラー表示とログの改善

## Phase 3: データソース拡張

- YouTube API 連携の検討
- Steam 公開ページ・API 連携の検討
- Google Trends 連携の検討
- GitHub トレンド取得の検討
- Reddit 連携の検討

## 後回し

- Web UI
- スケジューラー
- ダッシュボード
- TikTok の完全自動取得
- Instagram / X の自動取得
- 課金機能
