# Source Catalog

| source        | 対応方針 | 取得方法候補           | 主な取得データ                   | 注意点                         |
| ------------- | -------- | ---------------------- | -------------------------------- | ------------------------------ |
| TikTok        | 初期対応 | 手動 CSV               | 動画、キーワード、音源、指標     | 完全自動取得やスクレイピングはしない |
| YouTube       | 将来対応 | API 候補               | 動画、再生数、コメント等         | API キーや quota の確認が必要  |
| Steam         | 将来対応 | 公開ページ・API 候補    | ゲーム名、タグ、レビュー、人気度 | SNS とは別 source として扱う   |
| Google Trends | 将来対応 | API/ライブラリ候補     | 検索トレンド                     | 利用規約と制限の確認が必要     |
| GitHub        | 将来対応 | API 候補               | repository、star、言語、topic    | 技術トレンド分析向け           |
| Instagram / X | 非対応   | 手動候補               | 投稿、ハッシュタグ、指標         | 自動取得は当面行わない         |

## 初期 MVP の対象

初期 MVP では TikTok のみを対象にする。ただし TikTok 側から自動取得せず、人が確認した情報を `data/raw/tiktok/` に CSV として配置して取り込む。

新しい source を追加する場合は、このファイルと `docs/data-schema.md` を更新する。
